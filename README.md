JS/HTML/CSS

El procesamiento del PDF puede tener errores de lectura y las sugerencias del analizador no sustituyen el criterio clínico. El usuario es totalmente responsable de cotejar los datos transcritos con el documento original antes de integrarlos a un expediente legal o tomar decisiones terapéuticas.
